# Sorted Lists

Sorted Lists is a simple API that can store sorted lists in database.

## Stack

- Python
- FastAPI
- Pydantic
- TinyDB

## How to launch?

1. Init venv

    ```bash
    python -m venv venv
    source venv/bin/activate
    ```

2. Install dependencies

    ```bash
    pip install -r requirements.txt
    ```

3. Run server

    ```bash
    uvicorn main:app --reload
    ```

    or

    ```bash
    python main.py
    ```
